# MongoDB Get Started Sample Applications

This repository contains sample applications for getting started with
the MongoDB client libraries. Each folder corresponds to a driver language and
includes its own setup instructions.

## Available Client Libraries

- [Node.js](node/README.md)

## Prerequisites

Before running any sample application, complete the
[Atlas Get Started guide](https://www.mongodb.com/docs/get-started/)
to create a free Atlas deployment and load sample data.
